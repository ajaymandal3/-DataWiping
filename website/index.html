Frequently Asked Questions

Is it safe to wipe my main drive?

No, SecureWipe has safety checks to prevent wiping system drives. Only external drives and USB devices can be wiped.

Can data be recovered after wiping?

No, when using multiple-pass methods like Gutmann, data cannot be recovered even with forensic tools.

How long does wiping take?

Time depends on drive size and wipe method. 1-pass wipes are fastest; 35-pass Gutmann method is slowest.

Does it work with SSDs?

Yes, but TRIM commands make multiple-pass methods less effective. Use single-pass for SSDs.

Is this open source?

Yes! The entire project is open source under MIT license. Audit the code on GitHub.

Do you collect my data?

No analytics, no telemetry, no tracking. SecureWipe runs completely offline.
